Pozycja statyczna - to domyślna wartość położenia elementu. Można też używać go do usuwania pozycjonowania.

pozycjonowanie relatywne - przenosimy element względem bloku ustalonego dla tego elementu, ale pozostawiamy jego miejsce sprzed zmiany położenia w strukturze dokumentu. Przesuwamy go względem jego położenia spoczynkowego

pozycjonowanie absolutne - element zostaje przesunięty również względem bloku ustalonego ale nie zostaje jego miejsce sprzed zmiany położenia - element jest z niego wyjęty i zwalnia miejsce. Przesunięcie jest względem wybranych brzegów strony lub ewentualnie ramki na sztywno.

pozycjonowanie ustalone -fixed - działa jak absolutne, tylko że pozycja elementu jest zawsze obliczana względem krawędzi okna przeglądark oraz jest przesuwana wraz ze stroną
